# 🔬 Система сегментации и анализа руд (Ore Segmentation System)

Профессиональная платформа для автоматической семантической сегментации минералогического состава рудных шлифов с использованием глубокого обучения (U-Net + ResNet50) и классического компьютерного зрения.

## 📖 Описание проекта

Данная система представляет собой комплексное решение для анализа рудных шлифов, сочетающее:

- **Глубокое обучение**: U-Net с энкодером ResNet50 для точной семантической сегментации
- **Классическое CV**: Алгоритмы на базе OpenCV для быстрой предобработки и анализа
- **Интерактивный интерфейс**: Streamlit-приложение для удобной работы с данными
- **Автоматизацию**: Пайплайны генерации псевдомасок, патчинга и реорганизации данных

Система позволяет:

- Обучать модели сегментации на размеченных данных
- Анализировать панорамные изображения сверхвысокого разрешения
- Сравнивать результаты автоматической сегментации с экспертной разметкой
- Генерировать детальные PDF-отчёты с метриками качества (IoU, Dice)

## ✨ Возможности

### Основные функции

- **Семантическая сегментация**: Распознавание трёх классов минералов (тальк, тонкие срастания, крупные области)
- **Обработка панорам**: Автоматическое разбиение на патчи с перекрытием и бесшовная склейка
- **Интерактивная разметка**: Инструмент ручной разметки эксперта с поддержкой мультислойной аннотации
- **Сравнительный анализ**: Пространственные метрики совпадения (IoU, Dice Coefficient)
- **Автозагрузка моделей**: Мгновенная инициализация предобученных весов при старте
- **Экспорт результатов**: Генерация профессиональных PDF-отчётов с кириллицей

### 🔧 Утилиты

- **Pseudo Labeling**: Генерация псевдомасок через кластеризацию признаков ResNet50 + K-Means
- **Patching**: Разбиение панорамных изображений на патчи (1024×1024) с настраиваемым overlap
- **Mass Mask Creator**: Пакетное создание масок через пороговую бинаризацию
- **Filtr**: Предобработка изображений (CLAHE, нормализация яркости, конвертация в grayscale)
- **Reorganize Data**: Автоматическое приведение структуры датасета к стандарту `images/masks`

## 🛠 Технологии

### Deep Learning

- **PyTorch** 2.0+ — фреймворк глубокого обучения
- **Segmentation Models PyTorch** — библиотека предобученных архитектур
- **TorchVision** — трансформации и предобученные энкодеры

### Computer Vision

- **OpenCV** — классические алгоритмы обработки изображений
- **Albumentations** — продвинутая аугментация данных
- **Pillow** — работа с изображениями
- **NumPy** — численные операции

### Web Interface

- **Streamlit** — интерактивный веб-интерфейс
- **Streamlit Drawable Canvas** — инструмент ручной разметки

### Data & Visualization

- **Pandas** — анализ данных
- **Matplotlib** — визуализация и генерация PDF
- **Scikit-learn** — кластеризация (K-Means) и метрики

### Utilities

- **Joblib** — сериализация моделей
- **ReportLab** (опционально) — альтернативная генерация PDF

## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/AinazK/nornickel-material-hack.git
```

### 2. Переход в директорию проекта

```bash
cd nornickel-material-hack
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Установка весов

```bash
python utils/download.py
```

### 5. Запуск приложения

```bash
streamlit run app.py
```


## 🎯 Что реализовано:

✅ End-to-End пайплайн: От сырых панорам до обученной нейросети и веб-интерфейса.

✅ Гибридный подход: Сравнение нейросетевого контура с экспертной оценкой.

✅ Интерактивный UI: 3 вкладки: Статистика данных, Обучение модели, Анализ шлифа (U-net vs Ручная разметка).

✅ Ручная разметка в браузере: Интеграция streamlit-drawable-canvas для создания Ground Truth экспертами-геологами прямо в интерфейсе.

✅ Аналитическая отчетность: Автоматический расчет процентного соотношения минералов и классификация типа руды (рядовая, оталькованная, труднообогатимая).

✅ Экспорт артефактов: Возможность скачивания обученных весов (best_model.pth) и карт сегментации прямо из интерфейса.
